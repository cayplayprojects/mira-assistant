# 🌟 MIRA — AI Desktop Assistant v13.0 "AURORA"

**MIRA** — современный AI-ассистент для Windows с облачным (OpenRouter) и локальным (Ollama) ИИ. Управление ПК голосом и текстом, запуск приложений, поиск, сценарии, заметки и мониторинг системы.

## ✨ Возможности

*   🧠 **Двойной AI:** Облачный ИИ (OpenRouter — Gemini, Claude, GPT, Llama, DeepSeek) + локальный ИИ (Ollama)
*   🖥️ **Запуск приложений:** Запускает любые приложения и игры (Steam, Epic, Discord, VS Code и др.)

*   📊 **Мониторинг системы:** CPU, RAM, Disk в реальном времени
*   🔢 **Калькулятор:** "Сколько будет 25*4?"
*   ⏰ **Время и дата:** "Который час?", "Какое сегодня число?"

## 📥 Установка и Запуск

Для работы потребуется **Python 3.10+** и **Windows**.

### Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

> ⚠️ **Важно для PyAudio:** Если при установке ошибки:
> ```bash
> pip install pipwin
> pipwin install pyaudio
> ```

### Шаг 2: Установка AI (на выбор)

#### Вариант A: OpenRouter (облако, рекомендуется)
1. Зарегистрируйтесь на [OpenRouter](https://openrouter.ai)
2. Пополните баланс (есть бесплатные модели)
3. Скопируйте API-ключ
4. В MIRA: **Settings → OpenRouter API Key** → вставьте ключ

#### Вариант B: Ollama (локально)
1. Установите Ollama: [https://ollama.com/download](https://ollama.com/download)
2. Скачайте модель:
   ```bash
   ollama pull qwen3:1.7b
   ```
3. В MIRA: **Settings → Ollama (Local)**

### Шаг 3: Запуск

```bash
python mira.py
```

## ⚙️ Конфигурация (mira_config.json)

Файл создаётся автоматически при первом запуске.

### AI Провайдеры
```json
{
    "ai_provider": "openrouter",
    "openrouter": {
        "api_key": "sk-or-...",
        "model": "google/gemini-2.0-flash-001",
        "max_tokens": 2048,
        "temperature": 0.7
    },
    "ollama": {
        "base_url": "http://localhost:11434",
        "model": "qwen3:1.7b"
    }
}
```

### Модели OpenRouter
- `google/gemini-2.0-flash-001`
- `anthropic/claude-3.5-sonnet`
- `openai/gpt-4o-mini`
- `meta-llama/llama-3.3-70b-instruct`
- `deepseek/deepseek-chat-v3-0324`
- `qwen/qwen3-30b-a3b`

### Псевдонимы (Aliases)
```json
"aliases": {
    "стим": "steam.exe",
    "steam": "steam.exe",
    "блокнот": "notepad.exe",
    "фотошоп": "Photoshop.exe",
    "моя игра": "C:\\Games\\MyGame\\game.exe"
}
```

### Поисковая система
```json
"default_search": "yandex"
```
Доступны: `yandex`, `google`, `duckduckgo`, `youtube`

## ❓ FAQ

**Q: Как сменить AI модель?**
A: **Settings** → выберите провайдер → выберите модель из списка.

**Q: Не работает голос.**
A: Проверьте микрофон в настройках Windows. Нажмите 🎤 и говорите — статус "Listening..." подтвердит запись.

**Q: Как добавить своё приложение?**
A: Откройте `mira_config.json` → раздел `"aliases"` → добавьте `"название": "путь/к/exe.exe"`.

**Q: Как свернуть в трей?**
A: Нажмите **✕** — MIRA свернётся в трей. Двойной клик по иконке — восстановление.
