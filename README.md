# 🌟 MIRA — Виртуальный ИИ-Ассистент

**MIRA** — это современный, полноэкранный виртуальный ассистент с искусственным интеллектом, работающий локально на вашем компьютере. Создан для полного контроля над системой, автоматизации рутинных задач и быстрого поиска информации в стиле "Джарвиса".

> 🛡️ **Безопасность и Приватность:** Вся обработка нейросетью происходит **локально** на вашем железе (через Ollama). Никаких ключей API, никакой передачи данных в облако.

---

## ✨ Возможности

*   🧠 **Локальный AI:** Отвечает на вопросы, генерирует код, помогает с задачами (на базе LLM).
*   🖥️ **Умное управление ПК:** Запускает любые приложения, игры (Steam, Epic и др.) и файлы по голосовой команде или тексту.
*   🎤 **Голосовое управление:** Полноценный голосовой ввод и озвучка ответов.
*   📜 **Сценарии:** Создавайте макросы (например, "Режим стрима": запускает OBS, Discord, игру).
*   🌐 **Поиск в интернете:** Мгновенный поиск через браузер по команде "Найди в интернете...".
*   🎨 **Fluent UI:** Современный темный дизайн, адаптивный интерфейс, анимации.
*   📇 **Контакты:** Быстрый доступ к вашим соцсетям и мессенджерам.

##  Установка и Запуск

Для работы программы потребуется **Python 3.10** или выше.

### Шаг 1: Установка зависимостей

Откройте терминал (командную строку) в папке с проектом и выполните:

```bash
pip install -r requirements.txt
```

> ⚠️ **Важно для Windows (PyAudio):** Если при установке `pyaudio` возникают ошибки, установите его так:
> ```bash
> pip install pipwin
> pipwin install pyaudio
> ```

### Шаг 2: Установка Локального AI (Ollama)

MIRA использует **Ollama** для запуска нейросети.

1.  Скачайте и установите Ollama с официального сайта: [https://ollama.com/download](https://ollama.com/download)
2.  После установки запустите терминал и выполните команду для скачивания модели (занимает ~1-10 ГБ):
    ```bash
    ollama pull qwen3:3.5b
    ```
    *(Эта модель легкая, быстрая и отлично понимает русский язык).*

### Шаг 3: Запуск МИРА

Убедитесь, что Ollama работает в фоне (должна быть иконка в трее), и запустите ассистента:

```bash
python mira.py
```

При первом запуске автоматически создастся файл `mira_config.json` с настройками.

## 🗣️ Как пользоваться

Введите запрос в поле ввода или нажмите кнопку микрофона.

### Примеры команд:

| Тип | Команда | Результат |
| :--- | :--- | :--- |
| ️ **Запуск** | `Открой калькулятор` | Открывает `calc.exe` |
| 🎮 **Игры** | `Запусти Steam` или `Открой Minecraft` | Находит и запускает игру |
| 🌐 **Поиск** | `Найди в интернете погоду` | Открывает браузер с поиском |
|  **Сценарии** | `Создай сценарий: Утро` | Открывает мастер создания |
| 💬 **Вопрос** | `Кто ты и что умеешь?` | Отвечает нейросеть |

###  Сценарии (Автоматизация)

Вы можете автоматизировать запуск сразу нескольких приложений.
1. Скажите: *"Создай сценарий: Игры"*
2. В открывшемся окне пропишите команды (каждая с новой строки):
   ```text
   steam.exe
   discord.exe
   obs64.exe
   ```
3. Нажмите "Сохранить". Теперь командой *"Запусти сценарий: Игры"* вы откроете всё сразу.

## ⚙️ Настройка (mira_config.json)

Вы можете вручную редактировать файл `mira_config.json`, если программа не находит нужное приложение.

*   **Aliases (Псевдонимы):** Добавьте новое название для программы.
    ```json
    "aliases": {
        "моя игра": "C:\\Games\\MyGame\\game.exe"
    }
    ```
*   **Поисковики:** Измените дефолтный движок поиска (`yandex`, `google`).

## ❓ Частые вопросы (FAQ)

**Q: МИРА пишет "Ollama не найдена".**
**A:** Убедитесь, что вы установили [Ollama](https://ollama.com/) и она запущена (иконка 🦙 в трее). Выполните `ollama list` в терминале, чтобы проверить.

**Q: Не работает микрофон.**
**A:** Проверьте настройки конфиденциальности Windows: *Параметры -> Конфиденциальность -> Микрофон -> Разрешить приложениям доступ*.

**Q: Программа не находит игру.**
**A:** Откройте `mira_config.json` и добавьте путь к `.exe` файлу игры в раздел `"aliases"`.
