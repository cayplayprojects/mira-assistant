# 🌟 MIRA — Виртуальный ИИ-Ассистент v13.3 "NEBULA"

**MIRA** — это современный, полноэкранный виртуальный ассистент с искусственным интеллектом, работающий локально на вашем компьютере. Создан для полного контроля над системой, автоматизации рутинных задач и быстрого поиска информации.

> 🛡️ **Безопасность и Приватность:** Вся обработка нейросетью происходит **локально** на вашем железе (через Ollama). Никаких ключей API, никакой передачи данных в облако.

## ✨ Возможности

*   🧠 **Локальный AI:** Отвечает на вопросы, генерирует код, помогает с задачами (Qwen 3 на Ollama)
*   🖥️ **Умное управление ПК:** Запускает любые приложения, игры (Steam, Epic и др.) и файлы
*   🔒 **Закрытие приложений:** Закрывает программы по названию ("Закрой блокнот")
*   📋 **Мониторинг процессов:** Просмотр запущенных приложений в реальном времени
*   🎤 **Голосовое управление:** Полноценный голосовой ввод и озвучка ответов
*   📜 **Сценарии:** Создавайте макросы (например, "Режим стрима": запускает OBS, Discord, игру)
*   🌐 **Поиск в интернете:** Мгновенный поиск через Яндекс, Google или DuckDuckGo
